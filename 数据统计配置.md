# 网站访问量统计配置指南

## 快速开始

### 方案1：Vercel Analytics（推荐，最简单）

1. 安装依赖：
```bash
npm install @vercel/analytics
```

2. 已经配置完成！代码已经添加到 `app/layout.tsx`

3. 在Vercel Dashboard启用：
   - 进入项目设置
   - 点击 "Analytics" 标签
   - 点击 "Enable Web Analytics"

4. 查看数据：
   - 部署后等待几小时
   - 在Vercel项目页面查看Analytics面板

**优点**：
- ✅ 零配置
- ✅ 自动集成
- ✅ 实时数据
- ✅ 免费额度足够

**缺点**：
- ❌ 高级功能需要付费
- ❌ 数据留在Vercel

---

### 方案2：Google Analytics（免费且强大）

1. 创建Google Analytics账号：
   - 访问：https://analytics.google.com
   - 创建账号和属性
   - 获取测量ID（格式：G-XXXXXXXXXX）

2. 添加环境变量到 `.env.local`：
```bash
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX
```

3. 在 `app/layout.tsx` 中添加组件：
```tsx
import { GoogleAnalytics } from '@/components/google-analytics'

// 在 body 中添加：
<GoogleAnalytics gaId={process.env.NEXT_PUBLIC_GA_ID!} />
```

4. 在Vercel添加环境变量：
   - 项目设置 → Environment Variables
   - 添加 `NEXT_PUBLIC_GA_ID`

**查看数据**：
- https://analytics.google.com
- 实时报告、用户报告、流量来源等

**优点**：
- ✅ 完全免费
- ✅ 功能强大
- ✅ 详细的用户行为分析
- ✅ 支持自定义事件

**缺点**：
- ❌ 需要Google账号
- ❌ 中国访问较慢

---

### 方案3：Umami（开源、隐私友好）

1. 注册Umami Cloud账号：
   - 访问：https://umami.is
   - 或自己部署：https://github.com/umami-software/umami

2. 创建网站并获取Website ID

3. 添加环境变量：
```bash
NEXT_PUBLIC_UMAMI_WEBSITE_ID=your-website-id
```

4. 组件已经创建在 `components/umami-analytics.tsx`

5. 在 `app/layout.tsx` 中使用：
```tsx
import { UmamiAnalytics } from '@/components/umami-analytics'

// 在 body 中添加：
<UmamiAnalytics />
```

**优点**：
- ✅ 注重隐私
- ✅ 界面简洁
- ✅ 可自部署
- ✅ 无Cookie追踪

---

### 方案4：百度统计（国内推荐）

1. 注册百度统计：
   - https://tongji.baidu.com
   - 添加网站
   - 获取统计代码

2. 创建 `components/baidu-analytics.tsx`：
```tsx
"use client"

import Script from 'next/script'

export function BaiduAnalytics() {
  return (
    <Script
      id="baidu-analytics"
      strategy="afterInteractive"
      dangerouslySetInnerHTML={{
        __html: `
          var _hmt = _hmt || [];
          (function() {
            var hm = document.createElement("script");
            hm.src = "https://hm.baidu.com/hm.js?你的统计ID";
            var s = document.getElementsByTagName("script")[0]; 
            s.parentNode.insertBefore(hm, s);
          })();
        `,
      }}
    />
  )
}
```

3. 在layout中使用

**优点**：
- ✅ 国内访问快
- ✅ 中文界面
- ✅ 免费

## 多平台同时使用

可以同时使用多个分析工具：

```tsx
// app/layout.tsx
import { Analytics } from '@vercel/analytics/react'
import { GoogleAnalytics } from '@/components/google-analytics'
import { UmamiAnalytics } from '@/components/umami-analytics'

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        
        {/* Vercel Analytics */}
        <Analytics />
        
        {/* Google Analytics */}
        {process.env.NEXT_PUBLIC_GA_ID && (
          <GoogleAnalytics gaId={process.env.NEXT_PUBLIC_GA_ID} />
        )}
        
        {/* Umami */}
        <UmamiAnalytics />
      </body>
    </html>
  )
}
```

## 查看数据的关键指标

### 基础指标
- **访问量(PV)**：页面浏览次数
- **独立访客(UV)**：不重复的访问者
- **日活(DAU)**：每日活跃用户
- **月活(MAU)**：每月活跃用户

### 用户行为
- **平均停留时间**：用户在站停留多久
- **跳出率**：只访问一个页面就离开的比例
- **页面路径**：用户访问了哪些页面
- **设备分布**：手机/电脑/平板比例

### 流量来源
- **直接访问**：直接输入网址
- **搜索引擎**：从搜索来的
- **社交媒体**：从微博、微信等来的
- **外部链接**：从其他网站点击来的
